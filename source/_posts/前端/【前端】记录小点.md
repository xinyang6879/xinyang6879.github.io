### 如何固定安装依赖时所有依赖用到某个包的版本

使用`yarn`，在`package.json`中添加`resolutions`字段，并添加需要固定版本依赖的包名及版本号

``` json
{
    "dependencies": {
        "vue": '2.0.0',
        "vue-router": '3.0.0'
    },
    "resolutions": {
        "vue": '2.0.0',
    }
}
```

这样安装后，即使vue-router依赖的vue版本不是`2.0.0`,但可以强制其使用vue对应的版本
